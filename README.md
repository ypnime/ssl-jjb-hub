# ssl-jjb-hub

## 项目简介

本仓库用于归档与发布多个独立的 HTML 页面。这些页面内容不针对特定域名或网站，主要以通用信息展示、工具页面或静态演示为主。仓库本身是一个静态页面集合，便于浏览、引用或二次分发。

## 目录结构

```
ssl-jjb-hub/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：所有独立 HTML 页面均存放于此，每个页面可独立访问。
- `assets/`：页面引用的公共资源，按类型分目录存放。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可单独打开或部署。
- 页面之间无强依赖关系，但可能共享 `assets/` 下的资源。
- 归档内容可能随时间更新或清理，不保证长期不变。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接用浏览器打开 `pages/` 下的某个 HTML 文件即可查看。
3. 也可将整个仓库部署到任意静态托管服务（如 GitHub Pages、Netlify 等）进行在线访问。
